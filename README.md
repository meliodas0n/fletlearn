### How to install ?
#### 1. create a virtual environment
####    `python3 -m venv .venv`
#### 2. Activate the newly create environemnt
####    `source ./venv/bin/activate`
#### 3. Install poetry
#### `pip3 install poetry`
#### 4. Install required dependcies using poetry
#### `poetry install`
#### 5. Run the app
#### `python3 chrono`
